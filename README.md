# Linux Server Monitoring Script

This project provides a Bash script to monitor Linux server health.

It displays key system metrics including CPU usage, memory usage, disk usage, and running services.

---

## Features

* Monitor CPU usage
* Check memory utilization
* View disk space usage
* Display running services
* Show top CPU consuming processes

---

## Technologies Used

* Linux
* Bash Scripting

---

## How to Run

Make the script executable

chmod +x monitor.sh

Run the script

./monitor.sh
